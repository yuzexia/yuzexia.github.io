操作方法

字符串截取sub,substing,slice
  • 一个参数

sub,slice支持参数为正数和负数(等价于字符串长度+负数)，
substring只支持参数为正数，如果为负数会返回整个字符串

  • 两个参数

substr，第二个参数代表的是截取字符串的长度，第二个参数不能为负数;
substing，参数为负数会自动转换为0，并且数值小的为起点。
slice，第二个参数的数值小于第一个参数的数值返回空值。
